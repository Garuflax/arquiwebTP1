\begin{frame}
    \frametitle{Posicionamiento de resultados orgánicos}
    \begin{itemize}
    \item Se basa en indexación de las arañas web (Web Crawlers)
    \item[] \begin{itemize}
            \item Estas son aplicaciones que recorren las páginas web y almacenan las palabras clave relevantes
            \item Ejemplos: Googlebot y Bingbot
            \end{itemize}
    \item SEO busca optimizar la estructura y el contenido de una página web para facilitar el procesamiento a las arañas web
    \end{itemize}
    
\end{frame}

\begin{frame}
    \frametitle{Tipos de SEO}
    \begin{itemize}
    \item Interna (On-page SEO): mejoras en el contenido, mejoras técnicas en el código, accesibilidad, etc.
    \item Externa (Off-page SEO): mejorar la notoriedad de la web mediante referencias a ella
    \end{itemize}

    El análisis web, web analytics en ingles, es la medición, recopilación, análisis y generación de informes de datos web para comprender y optimizar el uso de la web.

La analítica web no es solo un proceso para medir el tráfico web, sino que puede utilizarse como una herramienta de marketing para evaluar y mejorar la eficacia del sitio web.

\begin{frame}
    \frametitle{Desarrollo de KPI}
    Un KPI es un indicador clave en el desempeño del negocio, el cual nos sirve para poder medir el éxito o el fracaso de nuestras acciones de marketing.
    
\end{frame}

\begin{frame}
    \frametitle{Recopilación de datos}
    
    Web server log file analysis
    \begin{itemize}
            \item Ventaja: Los datos están en los propios servidores de la empresa
            \item Ventaja: No hay llamadas a servidores externos que puedan reducir la velocidad de carga
            \item Principal problema: Caches
    \end{itemize}
    
\end{frame}

\begin{frame}
    \frametitle{Recopilación de datos}
    
    Page tagging
    \begin{itemize}
            \item Ventaja: Soluciona el problema de las páginas cache, que pueden representar hasta un tercio de todas las páginas vistas
            \item Ventaja: El page tagging puede informar información sobre eventos que no implican una solicitud al servidor web, ej: focus o clicks del mouse sobre algún boton
            \item Desventaja: Hay llamadas a servidores externos que puedan reducir la velocidad de carga
            \end{itemize}
    
\end{frame}

\begin{frame}
    \frametitle{Analítica fuera del sitio}
Métricas que no dependen del ingreso de un usuario al sitio.
Puede incluir medición de potencial audiencia (por ejemplo, viendo la cantidad de busquedas de googles relacionadas con tu negocio), también de la misma forma, potenciales competidores, actividades relacionadas en redes sociales, etc. 

Por ejemplo, Comscore te da:

Perfil demográfico: Información sobre la edad, género, clase social, provincia, estructura familiar?tanto de los usuarios que visitan una web como de los de la competencia.
Comparativa con los competidores: Informes sobre cómo evoluciona la audiencia de los diferentes sitios web del sector.
Audiencia compartida: indica el porcentaje de audiencia que compartimos con otros sitios web de nuestro mismo sector de actividad.

Silverbakk, hace un trackeo de redes sociales: Instagram, Facebook, Facebook Messenger, etc.
    
\end{frame}

\begin{frame}
    \frametitle{Analítica en el sitio}
Aquellas obtenidas cuando un visitante llega al sitio web.
Por lo general, estos datos se comparan con los indicadores clave de rendimiento y se utilizan para mejorar la respuesta de la audiencia de un sitio web o una campaña de marketing. Google Analytics y Adobe Analytics son los servicios más utilizados.
Con diferencia, la herramienta más popular es Google Analytics, la cual es usada por más del 80\% de los administradores de páginas web que hacen análisis. 
    
\end{frame}

\begin{frame}
    \frametitle{Google Analytics}
Herramienta de analítica web de la empresa Google lanzada hace ya 15 años.
Ventajas:
\begin{itemize}
\item Se integra con las plataformas y productos de marketing y publicidad de Google (incluidos Google Ads, Search Console y Data Studio), lo que lo convierte en una opción popular para cualquiera que utilice varias herramientas de Google.
\item Hay dos versiones, una paga y una gratuita. Para las empresas medianas y chicas alcanza y sobra con la versión gratuita.
\item Es simple de navegar e intuitivo.
\end{itemize}

Desventajas:
\begin{itemize}
\item Es un poco abrumadora la cantidad de dashboards, settings, metricas que hay. Lo cual puede asustar a usuarios principiantes y conlleva una curva de aprendizaje algo pronunciada.

\item En general, la versión gratuita sirve, pero si tenes un trafico web alto y querés pasar a la version premium, tenés que pagar aprox 200 dolares.

\end{itemize}
    
\end{frame}
